## 프로젝트 소개

2025년 설날을 맞아 진행하는 **무료 어린이 코딩 교육 챌린지**입니다.
아이들이 옥토스튜디오(Octo Studio)를 사용해 디지털 연하장을 만들고, 가족들에게 선물하는 프로젝트입니다.

- **주최**: 해달에듀
- **협력**: EBS 이솦
- **기간**: 2025년 1월 17일 ~ 2월 27일
- **대상**: 초등학생 및 유아

### 주요 기능
- 챌린지 소개 및 일정 안내
- 무료 참가 신청 폼
- 교육 영상 안내
- 시상 정보

## 웹사이트

`index.html` - 챌린지 랜딩 페이지
- URL: https://haedalprogramming.github.io/codingChallenge/
- Tailwind CSS 기반 반응형 디자인
- 참가 신청 폼 (Google Sheets + Zapier 연동)

## curri 폴더 (교육 커리큘럼)

옥토스튜디오를 활용한 3강 속성 교육 과정 자료입니다.

### 커리큘럼 구성

| 파일 | 설명 |
|------|------|
| `curri.md` | 전체 커리큘럼 개요 (3강 구성) |
| `video_overview_1.md` | **[영상 1] 기본편** - 앱 설치 ~ 캐릭터 만들기 ~ 기초 코딩 |
| `video_overview_2.md` | **[영상 2] 심화편** - 음성 녹음 & 센서 활용 인터랙션 |
| `video_overview_3.md` | **[영상 3] 실전편** - 최종 조립 & 촬영 노하우 |
| `slide_plan_*.md` | 각 영상별 슬라이드 기획안 |
| `slide_*.gs` | Google Slides 자동 생성 스크립트 |

### 교육 목표
1. **영상 1 (기본편)**: 10분 만에 내 얼굴이 춤춘다! - 설치, 캐릭터 생성, 기초 이동 코딩
2. **영상 2 (심화편)**: 흔들면 복이 우수수! - 음성 녹음, 터치/센서 반응 구현
3. **영상 3 (실전편)**: 조회수 폭발! - 최종 작품 조립, 촬영 및 업로드 가이드
